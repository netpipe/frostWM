Introduction

The Gnopernicus application enables users with limited vision, or no vision, to use the GNOME Desktop and associated applications. Gnopernicus provides the following functionality:

Screen reader

The screen reader enables non-visual access to standard applications in the GNOME Desktop, using speech and Braille output.

Magnifier

The magnifier provides automated focus tracking and fullscreen magnification to aid low-vision users.

For more information about Gnopernicus, visit the following website: http://www.baum.ro/gnopernicus

Getting Started

To Start Gnopernicus

You can start Gnopernicus in the following ways:

Applications menu

Choose Accessibility->Screen Reader and Magnifier.

Command line

Execute the following command: gnopernicus

You can use the -d option to specify default settings.

When You Start Gnopernicus

When you start Gnopernicus, the Gnopernicus menu window appears:

Figure 1. Gnopernicus Menu Window

The Gnopernicus menu window contains the following items:

Startup Mode

The Startup Mode dialog enables you to choose the functionality that you want Gnopernicus to provide when you launch the application. You can also apply functionality changes while you are using the application.

Preferences

You can adjust the settings for speech output, magnifier behavior, Braille output, Braille monitor, mouse behavior, and key mapping.

Default Preferences

You can click on this item at any time to load the default settings for Gnopernicus.

Minimize

You can click on this item to minimize the Gnopernicus menu window.

Help

You can click on this item to open the Help manual for Gnopernicus.

About

You can click on this item to read developer information about the Gnopernicus application.

Quit

You can click on this item to exit Gnopernicus at any time.

Usage

Gnopernicus functions are mapped onto keys on the numeric keypad. Because the number of keys on the numeric keypad are limited, the functions are assigned in layers. Any key can have multiple functions, depending on which layer is active. You can use the assigned layers of the Gnopernicus numeric keypad to work with the different functional areas of the application. The numeric keypad codes are written as follows: LayernnKeynn, where Layernn represents the number of the keypad layer, and Keynn represents the key value. The following sections include some useful numeric keypad assigned actions.

You can modify the defined actions for the numeric keypad layers, and you can also assign a keystroke sequence to any key on your keyboard, using the following modifiers:

  • Control

  • Alt

  • Shift

See the section called “Command Mapping With Layer Keys” for more information about numeric keypad actions.

To Switch Between Numeric Keypad Layers

To switch between numeric keypad layers, perform the following actions:

  1. Switch on the Num Lock.

  2. Press the 0 key on the numeric keypad.

  3. Press the key on the numeric keypad that corresponds to the layer you want activate, or press the Delete key for layer 10. You must perform this action within five seconds of the first action. If you do not press the layer number within five seconds, then Gnopernicus does not change the current layer.

To Modify Defined Layer Keys

Perform the following steps to modify layer keys that are already defined:

  1. Click on Command Mapping in the Gnopernicus menu window.

  2. Open the Layers tab in the Command Mapping dialog.

  3. Select the layer code you want to modify.

  4. Click Modify.

  5. Make your modifications.

  6. Click OK.

To Define Your Own Keystrokes

Perform the following steps to assign your own keystrokes to keys on your keyboard:

  1. Click on Command Mapping in the Gnopernicus menu window.

  2. Open the User Defined tab in the Command Mapping dialog.

  3. Click Add.

  4. Create your keystroke sequence by selecting the modifier options.

  5. Assign a key from your keyboard.

  6. Click OK

To Activate Speech Output

Speech output is normally active by default when Gnopernicus starts. However, if there is no speech output, click on Startup Mode in the Gnopernicus menu window, then select the Speech option in the Startup Mode dialog.

Some useful assigned actions on the numeric keypad that you can use to work with the Speech output feature are as follows:

Numeric Keypad Code

Assigned Action

L08K13

Pause or continue reading a large chunk of text.

L08K01

Decrease the speech volume.

L00K05

Repeat last utterance. To spell out characters, press Ctrl-Left Arrow.

L03K04

Provide speech output of the complete contents of a window through the heirarchy of the contents. This is called a logical review.

L03K05

Provide speech output of the complete contents of a window flattening the heirarchy in a left-to-right top-to-bottom way. This is called a spatial review.

To Start Braille Monitor

You can start Braille Monitor in the following ways:

Gnopernicus Menu Window

Open the Startup Mode dialog, then select the Braille Monitor option.

Command line

Execute the following command: brlmonitor

You can specify the following options with the brlmonitor command:

[-column | -c]

Number of columns on display

[-line | -l]

Number of lines on display

[-mode | -m]

Specifies the font: [normal | braille | dual]

[-port | -p]

UDP port number

To Activate Braille Output

Click on Braille in the Gnopernicus menu window, then select the Braille option in the Startup Mode dialog.

To Activate the Magnifier

Click on Magnifier in the Gnopernicus menu window, then select the Magnifier option in the Startup Mode dialog.

When the magnifier is activated, half of the screen appears normal size and the other half of the screen is magnified. The location of the cursor on the normal-size screen is indicated by a mobile crosshair on the magnified part of the screen.

Preferences

To control the behavior of the various functional areas in Gnopernicus, you select options in preference dialogs specific to each functional area. To open the Preferences menu window, click on Preferences in the Gnopernicus menu window.

Speech Preferences

Click on Speech to open the Speech Preferences dialog.

You can change the speech characteristics for punctuation reading, text echo, and several other parameters.

Braille Preferences

Click on Braille to open the Braille Preferences dialog.

You can change the following functionality:

  • Braille device

  • Translation table

  • Braille style

  • Cursor preferences

  • Braille key mapping

Magnifier Preferences

Click on Magnifier to open the Magnifier Preferences dialog.

You can change the following functionality:

  • Cursor settings

  • Crosshair settings

  • Zoomer list

Mouse Preferences

Click on Mouse to open the Mouse Preferences dialog.

You can select the following options for mouse behavior:

  • Take mouse at current object

  • Simulate click as current object

Braille Monitor Preferences

Click on Braille Monitor to open the Braille Monitor Preferences dialog.

You can change the following functionality:

  • Size

  • Font

  • Mode type

  • Position

  • Color

Command Mapping Preferences

Click on Command Mapping to open the Command Mapping dialog.

You can work with the following command mapping tabs:

  • Layers

  • User defined

Presentation Preferences

Click on Presentation to open the Presentation Preferences dialog.

You can select the following functionality:

  • Verbose

  • Default

Screen Review Preferences

Click on Screen Review to open the Screen Review Preferences dialog.

You can select the following functionality:

  • Horizontal flags

  • Vertical flags

Search Preferences

Click on Search to open the Search Preferences dialog.

You can search for an object on your screen, within the following parameters:

  • Window

  • Application

  • Desktop

  • Text

  • Attribute

  • Graphics
